## Idea / ایده

The main idea is to use division reminder. when division of number x is 0, it's divisible to that number. With that, we can add special string based on checking the condition with just one loop over the array. Also using 'continue' with separate 'if' statement will cause to check conditions early and stop checking others (althought we can use if else if too).

## Complexity / پیچیدگی

- Time / زمان: O(n)
- Space / حافظه: O(n)
